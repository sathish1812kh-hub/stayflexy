## 2. Gaps & Partial Feature Notes

- **Platform Telemetry Exposer (`Partial`)**: The OpenTelemetry tracer infrastructure and Jaeger export settings are complete. Express services expose the `/metrics` endpoint; however, in individual Express services, the `MetricsRegistry` and HTTP middleware need to be manually initialized and registered inside the service launchers to feed real HTTP execution metrics.
- **OTA sync providers (`Active`)**: Integrations with major channels (Booking.com, Expedia) are designed via JSON sync layouts. Real external socket connections are simulated via connection jobs.
